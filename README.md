# 《祭父文》
维:慈父大人生于1931年6月17日，仙逝于2018年6月8日晨，享年87岁。

呜呼我父，出身贫寒，年幼丧母。

少时劳耕，不畏辛苦。

国难之际，毅然从伍。

战争期间，折冲御侮。

抗美援朝，荣立大功。

慈父一生，光明磊落。

作风正派，克己奉公。

高风亮节，有目共睹。

洁身自好，毕生如故。

工作之时，事业为重。

日夜操劳，奔波忙碌。

勤奋好学，喜读古书。

夫妻恩爱，家庭和睦。

教子有方，以身作则。

外表严厉，内心温和。

音容笑貌，历历在目。

天有不测，人有祸福。

晴天劈雳，慈父驾鹤。

举家同悲，天地挽歌。

慈父恩泽，万世不枯。

精神永在，浩气千古。

呜呼哀哉！伏维尚飨！
